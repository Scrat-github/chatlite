# ChatLite MVP 开发任务

## 项目
ChatLite - 小企业客服工具（Intercom 开源替代）

## 技术栈
- Next.js 14 + TypeScript + Tailwind CSS
- Supabase (PostgreSQL + Auth + Realtime)
- Stripe 支付

## MVP 功能（2 周完成）

### 核心功能
1. 实时聊天 Widget（网站嵌入）
2. 客服仪表盘（对话管理）
3. 用户认证（团队 + 坐席）
4. Stripe 订阅（$29/月，14 天试用）

### 开发步骤

**第 1-3 天：项目骨架**
- 创建基础布局
- 配置 Supabase
- 配置 Stripe

**第 4-7 天：聊天功能**
- Widget 组件
- Realtime 消息同步
- 客服仪表盘

**第 8-10 天：用户系统**
- 注册/登录
- 团队管理
- 坐席权限

**第 11-14 天：支付 + 测试**
- Stripe 集成
- 部署到 Vercel
- 测试和文档
