## ⚠️ 可能的问题

### 如果注册失败

1. Supabase 表结构是否正确
2. Row Level Security 是否启用
3. 浏览器控制台是否有错误
4. Supabase Logs 是否有错误

**常见错误**:
- `relation "workspaces" does not exist` → 表未创建
- `permission denied` → RLS 配置问题
- `Invalid API key` → 环境变量错误

### 如果登录失败

**检查点**:
1. 账号是否注册成功
2. 密码是否正确
3. Supabase Auth 是否配置
